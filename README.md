## Описание функционала

Приложение – цифровая витрина для малых музеев Ставрополья.  
Оно решает проблему: у маленьких музеев нет сайтов, их не видно в интернете. Сотрудники музеев могут сами добавлять информацию, а посетители – находить музеи на карте, подписываться на новости и отмечать посещения.

### Возможности для посетителей
- **Главная** – карточки всех музеев с фото, описанием и отметкой «Пушкинская карта».
- **Карта** (Яндекс.Карты) – все музеи с метками, при клике показываются экспонаты.
- **События** – анонсы выставок и лекций. Можно включить фильтр «Только из подписанных музеев».
- **Мой паспорт** – статистика посещённых музеев и список подписок. Здесь же вход в админку.
- **Подписка на музеи** – чтобы видеть события только по интересующим музеям.
- **Отметка посещения** – одним кликом на карточке музея.

### Возможности для сотрудников (админ-панель)
- Вход по паролю `admin123` (единый для всех).
- **Музеи:** добавлять, редактировать, удалять (название, адрес, координаты, описание, контакты, сайт, обложка, Пушкинская карта да/нет).
- **Галерея музея:** можно добавить несколько дополнительных фото (показываются в карточке при клике).
- **Экспонаты:** добавлять, редактировать, удалять (название, описание, фото).
- **События:** добавлять, редактировать, удалять (название, дата, описание, фото).
